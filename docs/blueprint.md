# **App Name**: BlockGuard

## Core Features:

- Smart Contract Upload: Enable users to upload .sol files for analysis, including version checks and syntax validation.
- Static Analysis Engine: Run automated security analysis to detect common Solidity vulnerabilities.
- Vulnerability Dashboard: Display detected vulnerabilities with severity levels, code references, and clear explanations.
- AI-Powered Explanation Layer: Convert raw tool output into plain-English summaries, risk impact, and suggested fixes using AI as a tool.
- Scan History: Store previous scans to compare changes and track security improvements over time, backed by the PostgreSQL database.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to convey security, trust, and a professional feel.
- Background color: Light blue (#E6E9F2), a very low saturation version of the primary hue, suitable for a light scheme.
- Accent color: Purple (#7E57C2), saturation 80%, lightness 60%, which provides a good contrast for key actions and alerts.
- Body and headline font: 'Inter', a sans-serif font providing a modern, clean, and easily readable design.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use minimalist, geometric icons representing security concepts.
- Subtle animations to highlight detected vulnerabilities.
